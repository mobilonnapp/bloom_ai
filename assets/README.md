# Assets

Place your app assets here:

## Required
- `images/icon.png` — App icon (1024×1024 recommended)
- `images/splash.png` — Splash screen image
- `images/adaptive-icon.png` — Android adaptive icon foreground (1024×1024)

## Quick placeholder
Run this command to generate placeholder assets using Expo's defaults:
```bash
npx expo install expo-asset
```

Or copy any PNG as a placeholder to get the app running:
```bash
cp any-image.png assets/images/icon.png
cp any-image.png assets/images/splash.png
cp any-image.png assets/images/adaptive-icon.png
```
